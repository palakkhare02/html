What are Attributes?

Attributes provide extra information about elements.

Common Attributes
class

Used for grouping elements:

<p class="text">Hello</p>

id

Unique identification:

<p id="para1">Hello</p>

style

Inline CSS:

<p style="color:red;">Red Text</p>

title

Tooltip text:

<p title="Tooltip">Hover me</p>
